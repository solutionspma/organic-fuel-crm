üöÄ Quick Setup Guide

To see the backend in action:
  1. Create .env file from .env.example
  2. Configure Supabase credentials (URL + Service Role Key)
  3. Configure Telnyx credentials (API Key + Phone Number)
  4. Run Supabase schema: supabase/schema.sql
  5. Deploy to Netlify: npx netlify deploy --prod
  6. Or test locally (if workspace npm resolved): npx netlify dev

See QUICKSTART.md for detailed instructions.
